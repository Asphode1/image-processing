0. Install [Node.js](https://nodejs.dev/en/) (if not yet installed)

1. Clone this repo

```
git clone git@github.com:Asphode1/image-processing.git
```

2. Install package

```
npm install
```

3. Change `IMAGE_SRC`, `IMAGE_DEST` and add more extensions to `IMAGE_EXT` if needed

4. Run script

```
node index.js
```
